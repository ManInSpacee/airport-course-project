# Логотипы авиакомпаний

Положи сюда PNG-файлы с именами по IATA коду:
- `SU.png` — Аэрофлот
- `U6.png` — Уральские авиалинии
- `DP.png` — Победа
- `S7.png` — S7 Airlines

Рекомендуемый размер: 64×64 или 128×128, прозрачный фон.

Если файла нет — компонент автоматически покажет код авиакомпании в круглом badge.
